# UsedTextBooks

Marketplace for university students to buy and sell textbooks.

## Tech Stack

Frontend
- Angular

Backend
- Spring Boot
- PostgresSQL

Architecture
- Microservices
- Kafka
- JWT Authentication

## Development Phases

Phase 1 - Backend CRUD APIs  
Phase 2 - Angular UI  
Phase 3 - User Authentication  
Phase 4 - JWT Security  
Phase 5 - Marketplace features  
Phase 6 - Microservices  
Phase 7 - Kafka communication
